Инструкция по установке

  1. Убедитесь, что у вас установлен Minecraft 1.21.4 Fabric
  2. Скачайте и установите Fabric API (обязательно! без него мод не запустится):
    Скачать Fabric API для 1.21.4
  3. Скачайте JellybeanVisuals 1.0.0 beta по кнопке выше
  4. Поместите оба .jar файла (Fabric API + JellybeanVisuals) в папку %appdata%\.minecraft\mods
  5. Запустите Minecraft с профилем Fabric 1.21.4
  6. Готово! Нажмите Right Shift в игре, чтобы открыть меню
